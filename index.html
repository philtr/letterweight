letterweight

Estimates the length of a string in em units. Useful to calculate the apparent length of a string when using a variable-width font.

The widths are completely arbitrary since I could not find a resource to give me the average widths for each letter across the most common fonts. If you care to calculate them yourself or if you feel a width is incorrect feel free to submit an issue or pull request.

Usage

In your Gemfile

gem 'letterweight', '0.1.4'

In your code:

"This is a sentence I'd like to measure".to_em # => 20.5425
"MMMMMMMMMM".to_em # => 10.0
